Gem Installation/Update Dialog Box

See Also 

This dialog box opens when you are installing or updating a new Ruby gem after clicking OK in the Gem Installation Settings dialog box. It shows you the progress of the download and any messages that come back during the process. Typically the messages indicate if the installation was or was not successful. Sometimes you might be prompted to enter more information.

  • Some gem downloads prompt you for information. You can type a response directly in the dialog. When you see a message asking for more information, type your response and press Enter.
