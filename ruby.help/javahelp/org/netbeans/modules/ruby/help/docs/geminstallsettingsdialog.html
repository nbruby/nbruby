Gem Installation Settings Dialog Box

See Also 

Use this dialog box to install a new Ruby gem. It opens when you choose an available gem in the Gem Manager's New Gems tab and click Install. Typically, you just click OK to install a gem when this dialog opens.

  • When you click OK, an Installation dialog box opens and shows you the progress of your download. In addition, you might be prompted by the plugin's download program to specify some information for the plugin, so keep an eye on the Installation dialog while the download is happening.

You can set the following options:

  • Name. Specifies the name of the gem. If you want to install a different gem, it's easier to close the dialog box, select another gem in the New Gems tab, and click Install.
  • Version. By default, the latest version of the gem is installed. You can use this drop-down list to choose an earlier version.
  • Dependencies. By default, any gems that this gem depends on will be installed also. If you prefer that they not be installed, select Exclude.
