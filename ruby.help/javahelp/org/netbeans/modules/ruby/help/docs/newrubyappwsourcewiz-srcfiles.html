New Ruby Project with Existing Sources Wizard: Existing Sources

See Also 

In this final panel of the New Ruby Project with Existing Sources wizard, you specify where the source files and test files are for the existing project. Click the Add Folders buttons next to the Source Folders list and the Test Folders list to navigate to and select the folders.

When you create the project, the folders are not copied, but stay in their original location. If, using the IDE, you create a new file in one of the folders, the file is stored in the original folder.

You get to this panel by choosing Next in the Name and Location panel of the New Ruby Application with Existing Sources wizard.
