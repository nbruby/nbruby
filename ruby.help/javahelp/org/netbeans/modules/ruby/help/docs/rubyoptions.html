Setting Ruby Options

Use the Options window to make the following settings. You open the Options window by choosing Tools > Options from the main menu (NetBeans > Preferences on the Mac).

Editor Settings

Select Editor in the Options window to access the following settings.

  • Formatting. Click the Formatting tab and select Ruby from the Language drop-down list to change tab, indentation, and margin settings. You also use this tab to specify whether to reformat comments.
  • Code Templates. You use the Code Templates tab to add and modify code templates for various language, script, and file types including CSS, HTML, Ruby, Ruby Comments, Ruby HTML (RHTML and ERB), Ruby Quoted String, Ruby Strings, and YAML. For more information, see Creating and Customizing Ruby Code Templates.
  • Hints. Click the Hints tab and select Ruby from the Language drop-down list to choose the hints to be displayed in the Source Editor. You can also specify whether a hint should be displayed as an error, a warning, or a warning on the current line.
  • Macros. Use the Macros tab to edit, add, and remove Source Editor macros. For more information, see the Macros tab section in Options Window: Editor.

For additional information about editing options, see Options Window: Editor.

Font and Color Settings

Select Fonts and Colors in the Options window to customize the font and color settings for the various categories of language elements. You can also use this panel to customize how the Source Editor highlights and annotates the source, and how the IDE indicates changed text when comparing file revisions. You can set up multiple profiles, which you can save and import into other versions and installations of the IDE. For more information, see Options Window: Fonts and Colors.

Keymaps

You use this panel in the Options window to set the keyboard shortcuts for the IDE. Just as with the Fonts and Colors panel, you can set up multiple profiles. For more information, see Options Window: Keymap.

Miscellaneous

By default, the Projects window shows the logical view of Rails projects. If you prefer the window to show the physical view instead of the logical view, you can select Show Logical Project View in the Ruby tab of the Miscellaneous panel. You must restart the IDE before the change takes effect. Note that you can also see a file view of your projects by choosing Window > Files.

For information about the other tabs in the Miscellaneous panel, see the following help topics.

  • Options Window: Miscellaneous: Diff.
  • Options Window: Miscellaneous: ToDo Tasks.
  • Options Window: Miscellaneous: Versioning.
